Ce que j'accompagne

Je suis convaincue qu’aucun être humain n’est figé. Que derrière chaque peur, chaque conflit, chaque hésitation ou chaque période de doute, il existe une possibilité de mouvement.

Au fil de mon parcours, j’ai compris que cette réalité ne concernait pas uniquement les individus. Les équipes aussi peuvent se retrouver bloquées. Les organisations aussi peuvent se figer, dans leurs habitudes, dans leurs relations, dans leur manière de fonctionner, dans leur capacité à évoluer.

C’est de cette conviction qu’est née l’approche Détak. Une approche qui consiste à identifier ce qui freine le mouvement, puis à créer les conditions pour qu’il puisse reprendre.

Ce qui a besoin d’être détaqué peut se trouver dans la structure. Dans les relations. Ou dans le potentiel des personnes qui composent l’organisation. C’est pourquoi j’interviens aujourd’hui à travers ces trois dimensions complémentaires : la structure, les relations et le développement. Trois portes d’entrée différentes, pour un même objectif : remettre du mouvement là où quelque chose s’est figé.
